# CPP01

C++ Module 01 - Memory allocation, pointers to members, references, switch statement

## Exercises

### ex00 - BraiiiiinnnzzzZ
Zombie class implementation with stack and heap allocation.
- `make` to compile
- `./zombie` to run

### ex01 - Moar brainz!
Creating a horde of Zombies using dynamic array allocation.
- `make` to compile
- `./zombie_horde` to run

### ex02 - HI THIS IS BRAIN
Understanding the difference between pointers and references.
- `make` to compile
- `./brain` to run

### ex03 - Unnecessary violence
HumanA and HumanB classes with Weapon - demonstrating references vs pointers.
- `make` to compile
- `./violence` to run

### ex04 - Sed is for losers
String replacement in files without using sed.
- `make` to compile
- `./replace <filename> <s1> <s2>` to run

### ex05 - Harl 2.0
Harl class using pointers to member functions.
- `make` to compile
- `./harl` to run

## Compilation
All exercises use C++98 standard with flags: `-Wall -Wextra -Werror`
